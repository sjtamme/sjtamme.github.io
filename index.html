U.S. Unemployment Rates 2001-2015

A choropleth map in Leaflet. The data for this map was parsed from CSV format into a JSON object. A UI slider allows users to sequence through temporal data attributes and update the thematic map.

Dinosaur Fossils in North America

A bivariate map showing dinosaur fossils from the Mesozoic Era. The map uses Leaflet circle markers to show which fossils were from the Triassic, Jurassic, and Cretaceous periods.

Median Rent of US Counties

A dynamically generated choropleth map in Leaflet. The map is full-screen, mobile-friendly, and has an accompanying legend.

Shoshone Falls Elevation

This project used GDAL and QGIS 3.16 to convert raster elevation units and create contours, displayed on a Mapbox web map with geolocation enabled. The map shows all 2,525 named waterfalls in the United States in addition to the elevation around Shoshone Falls.

Appalachian Trail

A map showing a georeferenced image and digitized data to display a portion of the Appalachain Trail.
